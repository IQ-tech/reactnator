# Reactnator

### Installing

In your project folder, type on console

`yarn add reactnator`

or you can use NPM aswell

`npm i reactnator`

### Dependencies

* React
* ReactDOM
* Recompose

### Api Reference

* total: 1,
* currentPage: 1,
* onChange: e => console.log(e),
* position: false
